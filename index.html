Projects

DealHuntr

Developed in a team of 3 over 2 weeks, DealHuntr is a web application designed to aggregate and display real-time product deals from multiple sources. Focused on building a responsive and interactive frontend using Hotwire (Turbo + Stimulus), Bootstrap 5, and SCSS to deliver seamless user experience.

Implemented clean UI components, mobile-first design, and consistent styling with Simple Form and Font Awesome. Collaborated closely on backend RESTful APIs built with Ruby on Rails, utilizing PostgreSQL for data storage and SQL for efficient querying. The application was containerized with Docker and deployed on Heroku for scalable production use.

Technologies: Ruby on Rails, PostgreSQL, Hotwire (Turbo + Stimulus), Bootstrap 5, SCSS, Docker, Heroku, Simple Form, Font Awesome

WorkTree

Designed and delivered in a focused 5-day sprint by a team of three, WorkTree is an innovative platform that streamlines the process of reserving office and coworking spaces. Leveraging Hotwire (Turbo + Stimulus) and Bootstrap 5, I developed a highly responsive and user-friendly frontend that enables real-time updates and smooth navigation without heavy JavaScript frameworks.

The backend, powered by Ruby on Rails and PostgreSQL, provides robust RESTful APIs for secure user authentication (via Devise) and efficient reservation management. Containerized with Docker and deployed on Heroku, the system is built for scalability and reliability, supporting seamless booking experiences for modern workplaces.

Technologies: Ruby on Rails, PostgreSQL, Hotwire (Turbo + Stimulus), Bootstrap 5, Docker, Heroku, Devise, RESTful APIs
